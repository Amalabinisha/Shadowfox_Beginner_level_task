# Shadowfox_Beginner_level_task
# Iris Dataset Visualization using Python

## Project Overview

This project demonstrates Exploratory Data Analysis (EDA) and data visualization using the Iris Dataset. It was completed as part of the **Beginner Level Data Science Internship** at **ShadowFox**.

The project uses **Pandas** for data handling, **Matplotlib** for basic visualizations, and **Seaborn** for advanced statistical visualizations.

---

## Features

- Load the Iris dataset using Scikit-learn
- Perform data preprocessing
- Display dataset summary and statistics
- Generate 12 Matplotlib visualizations
- Generate 15 Seaborn visualizations
- Save all plots automatically into organized folders
